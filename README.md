## Adding a file

House rule: **one file per tool** — inline CSS, inline JS, no external dependencies
(optional CDN fonts only). It must run offline from the file itself.

1. Drop `your-tool.html` at the repo root
2. Add a row to the table above
3. GitHub Pages serves it at `/your-tool.html` ✈️
